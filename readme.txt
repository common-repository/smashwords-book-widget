== Description ==
This widget displays one or more random book covers of a [Smashwords](http://smashwords.com "Smashwords") author. It can be used by authors to display their work, by fans or by Smashwords affiliate partners.
The author, affiliate and text can be configured. The books can be flipped through as a 'slideshow'.

**Please note**: This widget is developed and maintained by [UnleashYourAdventure](http://unleashyouradventure.com "UnleashYourAdventure"), a small publishing house for travel ebooks. We developed it for ourself and our affiliate partners. We offer it as Open Source to give something back to the community. If you find it useful feel free to use it as well!

**This plugin is not developed or supported by Smashwords.**

== Installation ==

1. Goto Plugins -> Add new -> Upload and upload SmashwordsBookWidget.zip
1. Activate the plugin through the 'Plugins' menu
1. Place the Widget somewhere on your page in Appearance -> Widgets
1. Edit the settings in the widget on the same page
